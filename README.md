#  Clone da Netflix – Projeto nº 21

Projeto desenvolvido pelo aluno **Wanderson Brito Santos**.  
Este é um projeto fictício que recria uma página inspirada na interface da **Netflix**, com o objetivo de treinar e aprimorar habilidades em desenvolvimento front-end e ferramentas de automação.

---

##  Tecnologias utilizadas

As seguintes ferramentas foram usadas durante o desenvolvimento do projeto:

- **HTML5**
- **CSS3 / SASS**
- **Gulp**
- **Git & GitHub**
- **Vercel** (deploy)
- **JavaScript** (quando necessário)

---

##  Objetivo do projeto

O propósito deste projeto é:

- Praticar a estruturação de páginas HTML;
- Aperfeiçoar o uso de **CSS/SASS** na estilização;
- Utilizar o **Gulp** para tarefas automáticas (compilação, minificação e otimização);
- Treinar o uso de **Git/GitHub** para versionamento;
- Realizar deploy usando **Vercel**;
- Reproduzir uma interface moderna e responsiva inspirada na Netflix.
